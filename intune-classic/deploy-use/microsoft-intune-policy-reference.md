# <a name="microsoft-intune-configuration-policy-reference"></a>Справочник по политике конфигурации Microsoft Intune

[!INCLUDE [classic-portal](../includes/classic-portal.md)]

Здесь приведены сведения, которые помогут вам определить политику конфигурации Microsoft Intune, необходимую для управления устройствами.

> [!TIP]
> Дополнительные сведения об использовании политик см. в статье [Управление параметрами и компонентами на устройствах с помощью политик Microsoft Intune](manage-settings-and-features-on-your-devices-with-microsoft-intune-policies.md).


## <a name="android-configuration-policies"></a>Политики конфигурации Android

|Имя политики|Обеспечиваемые возможности|
|---------------|------------------------|

Это полезно в том случае, если необходимая настройка недоступна в политике конфигурации.<br /><br />Дополнительные сведения см. в статье [Параметры политики Android в Microsoft Intune](android-policy-settings-in-microsoft-intune.md).|

|**Общая конфигурация (Android 4 и более поздние версии, Samsung KNOX Standard 4.0 и более поздние версии)**<br><br>**Общая конфигурация (Android for Work)**|Настройка параметров безопасности и функциональных параметров мобильных устройств.<br />Определение приложений, соответствующих или не соответствующих политике, и создание отчета при их использовании.<br />Настройка режима киоска, в котором можно заблокировать устройства и разрешить работу лишь определенных функций, например разрешить устройству запускать только одно приложение или отключить клавиши регулировки громкости.<br /><br />Дополнительные сведения см. в статье [Параметры политики Android в Microsoft Intune](android-policy-settings-in-microsoft-intune.md).|
|**Профиль сертификата PKCS #12 (PFX) (Android 4 и более поздние версии)**<br><br>**Профиль сертификата PKCS #12 (PFX) (Android for Work)**|Используйте этот профиль для создания и развертывания настроек PFX в ответ на запросы на сертификаты устройств.<br /><br />Дополнительные сведения см. в статье [Защита доступа к ресурсам с помощью профилей сертификатов в Microsoft Intune](secure-resource-access-with-certificate-profiles.md).|

Развертывая эти настройки, вы упрощаете подключение для конечных пользователей к их работе.<br /><br />Дополнительные сведения см. в статье [VPN-подключения в Microsoft Intune.md](vpn-connections-in-microsoft-intune.md).|
|**Профиль Wi-Fi (Android 4 и более поздней версии)**<br><br>**Профиль Wi-Fi (Android for Work)**|Настройте и выполните развертывание параметров беспроводной сети для пользователей в вашей организации.
