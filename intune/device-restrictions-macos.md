# <a name="macos-device-settings-to-allow-or-restrict-features-using-intune"></a>Параметры устройства с macOS, которые позволяют разрешить или ограничить некоторые функции через Intune

[!INCLUDE [azure_portal](./includes/azure_portal.md)]

В этой статье перечислены и описаны параметры, которыми можно управлять на устройствах с macOS. В решении по управлению мобильными устройствами (MDM) эти параметры позволяют разрешать или отключать использование функций, задавать правила для паролей, разрешать и ограничивать отдельные приложения и т. д.

Эти параметры можно добавить в профиль конфигурации устройства в Intune, а затем назначить или развернуть на устройствах c macOS.

## <a name="before-you-begin"></a>Подготовка к работе

[Создание профиля конфигурации устройства ограничения](device-restrictions-configure.md#create-the-profile).

## <a name="general"></a>Общие

- **Блокировать поиск определений**. Выберите **Блокировать**, чтобы пользователи не могли выделять слова и искать их определения на устройстве. Если выбрать вариант **Не настроено** (по умолчанию), доступ к функции поиска определения будет разрешен.
- **Блокировать диктофон**: выберите пункт **Блокировать**, чтобы пользователи не могли вводить текст с помощью голосового ввода. Если выбрать вариант **Не настроено** (по умолчанию), пользователь сможет использовать голосовой ввод.
- **Блокировать кэширование содержимого**: выберите **не настроен** (по умолчанию), чтобы включить кэширование содержимого. Кэширование содержимого хранит данные приложений, веб-обозревателя данных, файлы для загрузки и многое другое локально на устройстве. Выберите **блок** для предотвращения этих данных, хранящихся в кэше.

  Дополнительные сведения о кэширование содержимого в macOS, см. в разделе [управление кэширование содержимого на компьютере Mac](https://support.apple.com/guide/mac-help/manage-content-caching-on-mac-mchl3b6c3720/mac) (открывается другой веб-сайт).

  Данная функция применяется к:  
  - macOS 10.13 и более поздние версии

- **Отложить обновления программного обеспечения (только в защищенном режиме)**: Если значение **не настроен** (по умолчанию), обновления программного обеспечения, отображаются на устройстве как Apple их выпуска. Например если обновление macOS получает выпущен компанией Apple в определенный день, затем обновления естественным образом появится на устройстве с датой выпуска.

  **Включить** позволяет задержать, если обновления программного обеспечения, которые отображаются на устройствах из 0 – 90 дней. Этот параметр не контролирует, когда обновления или не установлены. 

  - **Задержка видимостью обновлений программного обеспечения**: Введите значение от 0 – 90 дней. Когда задержка истекает, пользователи получают уведомление о том, что необходимо обновить ОС до самой ранней версии, которая была доступна на момент активации задержки.

    Например, если обновление macOS можно найти в **1 января**, и **отложить видимость** присваивается **5 дней**, то обновление не отображается как доступное обновление на устройствах. На **шестой день** после выпуска, доступно обновление, что конечные пользователи могут установить его.

    Данная функция применяется к:  
    - macOS 10.13.4 и более поздние версии

## <a name="password"></a>Пароль

- **Пароль.** Пользователь **должен** ввести пароль для доступа к устройству. **Не настроено** (по умолчанию) не требует пароля и не требует каких-либо ограничений, например блокировка простых паролей или установка его минимальную длину.
  - **Требуемый тип пароля**. Укажите, может ли пароль быть только числовым или он должен быть буквенно-цифровым (содержать буквы и цифры). Этот параметр поддерживается только в Mac OS X версии 10.10.3 и более поздних версиях.
  - **Число символов, отличных от буквенно-цифровых, в пароле**. Указывает количество сложных символов, которые должны содержаться в пароле (от **0** до **4**).<br>Сложный символ — это специальный знак, например "**?**".
  - **Минимальная длина пароля**. Укажите минимальную длину пароля, который должен ввести пользователь (от **4** до **16** символов).
  - **Простые пароли**. Этот параметр разрешает использовать простые пароли, например **0000** или **1234**.
  - **Максимальный срок после блокировки экрана (в минутах), по истечении которого запрашивается пароль**. Укажите, как долго компьютер должен оставаться неактивным, прежде чем потребуется пароль для разблокировки.
  - **Максимальное время бездействия (в минутах), по истечении которого экран блокируется**. Укажите период бездействия компьютера, по истечении которого экран блокируется.
  - **Окончание срока действия пароля (в днях)**. Укажите, сколько дней должно пройти, прежде чем пользователь должен будет сменить пароль (от **1** до **255** дней).
  - **Запретить повторное использование предыдущих паролей**. Укажите количество предыдущих паролей, которые нельзя использовать повторно (от **1** до **24**).

- **Запретить пользователю Изменение секретного кода**: выберите **блок** остановить от изменений, добавляемых или удаляемых секретный код. Если выбрать вариант **Не настроено** (по умолчанию), можно будет добавлять, изменять и (или) удалять секретные коды.
- **Блокировать разблокирование с помощью отпечатков пальцев**. Выберите **Блокировать**, чтобы запретить использование отпечатка пальца для разблокировки устройства. Если выбран вариант **Не настроено** (по умолчанию), пользователям разрешено разблокировать устройство с помощью отпечатков пальцев.

- **Блокировать автозаполнение пароля**. Выберите **Блокировать**, чтобы запретить использование функции автозаполнения паролей в macOS. Выбор **блок** также имеет следующие последствия:

  - пользователям не предлагается использовать сохраненный пароль в Safari и в любых других приложениях;
  - отключается функция автоматических надежных паролей и пользователям не предлагаются надежные пароли.

  Если выбрано значение **Не настроено** (по умолчанию), эти функции разрешены.

- **Блокировать запросы близкого взаимодействия пароля**. Выберите **Блокировать**, чтобы устройство пользователя не запрашивало пароли с близлежащих устройство. Если выбрать вариант **Не настроено** (по умолчанию), можно будет выполнять такие запросы.

- **Блокировать общий доступ к паролям**. Выберите **Блокировать**, чтобы запретить обмен паролями между устройствами через AirDrop. Если выбрать вариант **Не настроено** (по умолчанию), общий доступ к паролям будет разрешен.

## <a name="built-in-apps"></a>Встроенные приложения

- **Блокировать автозаполнение в Safari**. Выберите **Блокировать**, чтобы отключить на устройстве функцию автозаполнения в браузере Safari. Если выбрать вариант **Не настроено** (по умолчанию), пользователи смогут изменять параметры автозаполнения в браузере.
- **Блокировать камеру**. Выберите **Блокировать**, чтобы запретить доступ к камере на устройстве. Если выбрать вариант **Не настроено** (по умолчанию), доступ к камере устройства будет разрешен.
- **Блокировать Apple Music**: Выберите **Блокировать**, чтобы перевести приложение "Музыка" в классический режим и отключить службу Music. Если выбрать вариант **Не настроено** (по умолчанию), можно будет использовать приложение Apple Music.
- **Результаты поиска в Интернете Spotlight блок**: **блок** останавливает Spotlight возвращали результатов путем поиска в Интернете. Если выбрать вариант **Не настроено** (по умолчанию), функция поиска Spotlight сможет подключаться к Интернету для получения дополнительных результатов.
- **Передача файлов блока с помощью iTunes**: **блок** отключает службы обмена файлами приложения. В macOS 10.13 и более поздние версии. **Не настроено** (по умолчанию) разрешает общий доступ к файлам приложения службы.

## <a name="restricted-apps"></a>Ограниченные приложения

В списке ограниченных приложений можно настроить один из следующих списков.

- **Запрещенные приложения**. Содержит приложения, не управляемые Intune, которые пользователям запрещено устанавливать и запускать. Пользователям не разрешается Установка запрещенных приложений, но если это так, оно будет отмечено для администратора.
- **Утвержденные приложения**. Содержит приложения, которые пользователям разрешено устанавливать. Пользователям запрещено устанавливать приложения, которых нет в списке. Приложения под управлением Intune автоматически разрешены. Пользователям не запрещено устанавливать приложения, которой нет в списке утвержденных. Но если это так, оно будет отмечено для администратора.

Чтобы настроить список, щелкните **Добавить**, укажите выбранное имя, при необходимости укажите издателя приложения, и ИД пакета приложения (например, *com.apple.calculator*).

## <a name="connected-devices"></a>Подключенные устройства

- **Блокировать AirDrop**: выберите пункт **Блокировать**, чтобы запретить использование функции AirDrop на устройстве. Если выбрать вариант **Не настроено** (по умолчанию), можно будет использовать функцию AirDrop для обмена содержимым с близлежащими устройствами.
- **Блок Apple Watch автоматической разблокировки**: **блок** запрещает разблокировки устройства macOS в их Apple Watch. **Не настроено** (по умолчанию) позволяет пользователям разблокировать свои устройства macOS с их Apple Watch.

## <a name="cloud-and-storage"></a>Облако и хранилище

- **Блокировать синхронизацию цепочки ключей с iCloud.** Выберите **Блокировать**, чтобы отключить синхронизацию в iCloud учетных данных, которые хранятся в цепочке ключей. Если выбрать вариант **Не настроено** (по умолчанию), пользователь сможет синхронизировать эти учетные данные.
- **Блокировать синхронизацию документов iCloud**: **блок** запрещает синхронизацию документы и данные iCloud. Значение **Не настроено** (по умолчанию) разрешает синхронизацию документов и пар "ключ — значение" в хранилище iCloud.
- **Блокировать Mail резервной копии в iCloud**: **блок** запрещает синхронизацию в приложение "Почта" macOS iCloud. **Не настроено** (по умолчанию) разрешает синхронизация почты в iCloud.
- **Блокировать обратитесь к резервной копии в iCloud**: **блок** предотвращает синхронизировать контакты устройства в iCloud. **Не настроено** (по умолчанию) разрешает синхронизацию контактов с помощью iCloud.
- **Блокировать календарь резервной копии в iCloud**: **блок** запрещает синхронизацию в приложение календарь macOS iCloud. **Не настроено** (по умолчанию) разрешает Синхронизация календаря в iCloud.
- **Блокировать напоминание резервной копии в iCloud**: **блок** запрещает синхронизацию в приложение macOS напоминания iCloud. **Не настроено** (по умолчанию) позволяет синхронизировать напоминания в iCloud.
- **Блокировать закладки резервной копии в iCloud**: **блок** предотвращает iCloud синхронизации устройств закладки. **Не настроено** (по умолчанию) позволяет синхронизировать закладки в iCloud.
- **Блокировать заметки о резервной копии в iCloud**: **блок** предотвращает iCloud синхронизации устройств заметки. **Не настроено** (по умолчанию) разрешает синхронизацию заметок в iCloud.

## <a name="domains"></a>Domains

- **URL-адрес домена электронной почты**: добавьте в список один или несколько URL-адресов. Когда пользователь получает электронное письмо с домена, отличного от настроенного вами, оно помечается как ненадежное в приложении почты MacOS.
