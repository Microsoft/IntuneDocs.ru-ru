# <a name="how-to-monitor-app-information-and-assignments-with-microsoft-intune"></a>Как отслеживать сведения о приложении и его назначения с помощью Microsoft Intune

[!INCLUDE [azure_portal](./includes/azure_portal.md)]

Служба Intune предоставляет ряд способов для отслеживания свойств приложений, которыми вы управляете, а также их состояний назначения.

1. Войдите на [портал Azure](https://portal.azure.com).
2. Выберите **Все службы** > **Intune**. Intune находится в группе **Мониторинг и управление**.
3. Выберите **Мобильные приложения**, а затем **Приложения** в группе **Управление**.
5. В списке приложений выберите приложение, которое нужно отслеживать. Появится колонка приложения со сведениями о состоянии устройства и пользователя.

## <a name="app-overview-blade"></a>Колонка обзора приложений

В колонке приложения можно просмотреть подробные сведения о состоянии приложения в вашей среде.

### <a name="essentials"></a>Основное
В разделе **Основные сведения** содержатся следующие сведения о приложении:

 | **Сведения о приложении**            | **Описание**                                                      |
|------------------------|------------------------------------------------------------------|
| **Издатель**          | Издатель приложения.                                            |
| **Операционная система**   | Операционная система приложения (Windows, iOS, Android и т. д.) |
| **Создано**             | Дата и время создания этой версии.                         |
| **Назначено**           | **Да** или **Нет**, если приложение было назначено.                  |

### <a name="device-and-user-status-graphs"></a>Графы состояния устройства и пользователя
На графах представлены указанные ниже значения.

| **Состояние устройства**       | **Описание**                                       |
|-----------------------|-------------------------------------------------------|
| **Установлено**         | Количество установленных приложений.                         |
| **Не установлено**     | Количество неустановленных приложений.                     |
| **Сбой**            | Количество неудачных установок.                   |
| **Ожидается установка**   | Количество приложений в процессе установки. |
| **Неприменимо**           | Количество приложений с состоянием "Неприменимо".            |

### <a name="device-install-status"></a>Состояние установки на устройствах

Список состояний устройств отображается при выборе элемента **Состояние установки на устройствах** в разделе **Монитор**. В таблице сведений содержатся указанные ниже столбцы.

| **Столбец устройства**      | **Описание**                                                                                                                                                                                                                                            |
|----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Имя устройства**      | Имя устройства на платформах, допускающих именование устройства. На других платформах Intune создает имя из других свойств. Этот атрибут может быть доступен не для всех устройств.                                                                       |
| **Имя пользователя**        | Имя пользователя.                                                                                                                                                                                                                                      |
| **Платформа**         | Операционная система устройства (Windows, iOS, Android и т. д.)                                                                                                                                                                                           |
| **Версия**          | Номер версии приложения. Для бизнес-приложений отображается полный номер версии. Полный номер версии определяет конкретный выпуск приложения. Номер имеет следующий вид: _версия_(_сборка_). Пример: 2.2(2.2.17560800) |
| **Состояние**           | Состояние приложения.                                                                                                                                                                                                                                     |
| **Сведения о состоянии**   | Сведения о состоянии.                                                                                                                                                                                                                                     |
| **Последняя запись после изменения**    | Дата последней синхронизации устройства с Intune.                                                                                                                                                                                                                  |


### <a name="user-install-status"></a>Состояние установки пользователя

Список состояний пользователей отображается при выборе элемента **Состояние установки для пользователей** в разделе **Монитор**. В таблице сведений содержатся указанные ниже столбцы.

| **Столбец пользователя**     | **Описание**                           |
|---------------------|-------------------------------------------|
| **Имя**            | Имя пользователя в Azure Active Directory.         |
| **Имя пользователя**       | Уникальное имя пользователя.              |
| **Установки**   | Количество установок, используемых пользователем. |
| **Сбои**        | Количество неудачных установок, выполненных пользователем.     |
| **Не установлено**   | Количество приложений, не установленных пользователем. |
