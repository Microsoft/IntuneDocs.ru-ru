# <a name="remove-devices-by-using-factory-reset-or-remove-company-data"></a>Очистка устройств с помощью сброса параметров до заводских настроек или удаления данных организации

[!INCLUDE[azure_portal](./includes/azure_portal.md)]

Устройства, которые вам больше не нужны, используются с другой целью или не используются вообще, можно удалить из Intune. Это можно сделать, выполнив команду **удаления данных организации** или **сброса параметров до заводских настроек**. Кроме того, пользователи могут запускать удаленную команду с корпоративного портала Intune для личных устройств, зарегистрированных в Intune.

> [!NOTE]
> Прежде чем удалить пользователя из Azure Active Directory, выполните команду **сброса параметров** или **удаления данных организации** для всех устройств, связанных с этим пользователем. Если удалить пользователей с управляемыми устройствами из Azure Active Directory, Intune больше не сможет выполнять сброс параметров или удалять данных организации на этих устройствах.

## <a name="factory-reset"></a>Сброс параметров к заводским значениям

**Сброс параметров до заводских настроек** восстанавливает заданные по умолчанию заводские настройки, удаляя все корпоративные и пользовательские данные и параметры с устройства. Устройство удаляется из системы управления Intune. Сброс параметров до заводских настроек удобен для сброса устройства перед передачей его новому пользователю, а также в случае утери или кражи устройства. Соблюдайте осторожность при выборе сброса параметров до заводских настроек. Восстановить данные на устройстве будет невозможно.

### <a name="to-factory-reset-a-device"></a>Сброс параметров до заводских настроек

1. Войдите на [портал Azure](https://portal.azure.com).
2. Выберите **Больше служб** > **Мониторинг и управление** > **Intune**.
3. В колонке **Устройства и группы** выберите **Все устройства**.
4. Выберите имя устройства, для которого требуется восстановить заводские настройки.
5. В колонке с именем устройства выберите **Сброс параметров до заводских настроек** и нажмите кнопку **Да** для подтверждения.

Если устройство включено и подключено, выполнение команды сброса параметров до заводских настроек данных займет меньше 15 минут на всех типах устройств.

## <a name="remove-company-data"></a>Удаление данных организации

Команда **удаления данных организации** удаляет данные управляемого приложения (если применимо), параметры и профили электронной почты, которые были назначены с помощью Intune. При удалении данных организации личные данные пользователя остаются на устройстве. Устройство удаляется из системы управления Intune. ### <a name="to-remove-company-data"></a>Удаление данных организации

1. Войдите на [портал Azure](https://portal.azure.com).
2. Выберите **Больше служб** > **Мониторинг и управление** > **Intune**.
3. В колонке **Устройства и группы** выберите **Все устройства**.
4. Выберите имя устройства, с которого требуется удалить данные организации.
5. В колонке с именем устройства выберите **Удаление данных организации** и нажмите кнопку **Да** для подтверждения.

Если устройство включено и подключено, выполнение команды удаления данных займет меньше 15 минут на всех типах устройств.

## <a name="delete-devices-from-the-azure-active-directory-portal"></a>Удаление устройств на портале Azure Active Directory

Из-за проблем со связью или отсутствующих устройств вам может потребоваться удалить устройства из Azure Active Directory (AD). Команда удаления не удаляет устройство из системы управления, но команду **Удалить** можно использовать для удаления записей устройства из консоли Azure, которые известны как недоступные и вряд ли будут обмениваться данными с Azure снова.

1.  Войдите в [Azure Active Directory на портале Azure](http://aka.ms/accessaad) с помощью учетных данных администратора. Можно также войти на [портал Office 365](https://portal.office.com) и выбрать **Администрирование** &gt; **Azure AD** с помощью ссылки в левой части страницы.
3.  Создайте подписку Azure, если у вас ее нет. Если у вас платная учетная запись, кредитная карта или оплата не потребуются (щелкните ссылку **Зарегистрировать бесплатную подписку Azure Active Directory**).
4.  Выберите **Active Directory**, а затем выберите вашу организацию.
5.  Откройте вкладку **Пользователи**.
6.  Выберите пользователя, устройства которого требуется удалить.
7.  Выберите **Устройства**.
8.  Удалите соответствующие устройства, например неиспользуемые или имеющие неверные определения.
