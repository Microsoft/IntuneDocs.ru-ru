# <a name="remove-devices-by-using-factory-reset-or-remove-company-data"></a>Очистка устройств с помощью сброса параметров до заводских настроек или удаления данных организации

[!INCLUDE [azure_portal](./includes/azure_portal.md)]

Устройства, которые вам больше не нужны, используются с другой целью или не используются вообще, можно удалить из Intune. Это можно сделать с помощью действий **Удалить данные организации** или **Сбросить параметры до заводских настроек**. Кроме того, пользователи могут запускать удаленную команду с корпоративного портала Intune для личных устройств, зарегистрированных в Intune.

> [!NOTE]
> Прежде чем удалить пользователя из Azure Active Directory (Azure AD), выполните действия **Сбросить параметры до заводских настроек** или **Удалить данные организации** для всех устройств, связанных с этим пользователем. Если удалить пользователей с управляемыми устройствами из Azure AD, Intune больше не сможет выполнять сброс параметров или удалять данные организации на этих устройствах.

## <a name="factory-reset"></a>Сброс параметров к заводским значениям

Действие **Сбросить параметры до заводских настроек** восстанавливает на устройстве заводские настройки. При этом удаляются все данные и настройки организации и пользователя. Устройство удаляется из системы управления Intune. Сброс параметров до заводских настроек удобен для сброса параметров устройства перед передачей его новому пользователю, а также в случае утери или кражи устройства. Соблюдайте осторожность при выборе действия **Сбросить параметры до заводских настроек**. Восстановить данные на устройстве будет невозможно.

### <a name="factory-reset-a-device"></a>Сброс параметров устройства до заводских настроек

1. Войдите на [портал Azure](https://portal.azure.com).
2. Выберите **Все службы**, отфильтруйте список по **Intune** и выберите **Microsoft Intune**.
3. Выберите **Устройства** > **Все устройства**.
4. Выберите имя устройства, для которого требуется восстановить заводские настройки.
5. В области, где отображается имя устройства, выберите **Сбросить параметры до заводских настроек**.
6. Для Windows 10 версии 1709 или более поздних версий можно также применить параметр **Сохранить состояние и учетную запись регистрации**. 
    
    |Сохраняется во время сброса параметров до заводских настроек|Не сохраняется|
    | -------------|------------|
    |учетные записи пользователей, связанные с устройством;|файлы пользователя;|
    |состояние компьютера \(присоединение к домену, AADJ);| приложения, установленные пользователем \(магазин и приложения Win32);|
    |регистрация управления мобильными устройствами (MDM);|параметры устройства, отличные от заданных по умолчанию.|
    |приложения, установленные изготовителем оборудования \(магазин и приложения Win32);||
    |Профиль пользователя||
    |данные пользователя вне профиля пользователя;||
    |автоматический вход пользователя.|| 
    
         
7. Нажмите кнопку **Да**, чтобы подтвердить сброс до заводских настроек.

Если устройство включено и подключено, выполнение действия **Сбросить параметры до заводских настроек** займет меньше 15 минут на всех типах устройств.

## <a name="remove-company-data"></a>Удаление данных организации

Действие **Удалить данные организации** удаляет данные управляемого приложения (если применимо), параметры и профили электронной почты, которые были назначены с помощью Intune. Устройство удаляется из системы управления Intune. Это происходит всякий раз, когда устройство регистрируется и получает удаленное действие **Удаление данных компании**.

Действие **Удалить данные организации** оставляет личные данные пользователя на устройстве.

### <a name="remove-company-data"></a>Удаление данных организации

1. Войдите в [Intune на портале Azure](https://aka.ms/intuneportal).
2. На панели **Устройства** выберите **Все устройства**.
3. Выберите имя устройства, с которого требуется удалить данные организации.
4. В области, где отображается имя устройства, выберите **Удалить данные организации**. Нажмите кнопку **Да** для подтверждения.

Если устройство включено и подключено, выполнение действия **Удалить данные организации** займет меньше 15 минут на всех типах устройств.

## <a name="delete-devices-from-the-intune-portal"></a>Удаление устройств с портала Intune

Удалить устройства с портала Intune можно в соответствующей области. При следующей регистрации устройства все данные компании на нем будут удалены.

1. Войдите в [Intune на портале Azure](https://aka.ms/intuneportal).
2. Выберите **Устройства** > **Все устройства** > удаляемое устройство > **Удалить**.

## <a name="delete-devices-from-the-azure-active-directory-portal"></a>Удаление устройств на портале Azure Active Directory

Из-за проблем со связью или отсутствующих устройств вам может потребоваться удалить устройства из Azure AD. Действие **Удалить** удаляет записи устройства на портале Azure, которые известны как недоступные и вряд ли будут обмениваться данными с Azure снова. Действие **Удалить** не удаляет устройство из системы управления.

1.  Войдите в [Azure Active Directory на портале Azure](http://aka.ms/accessaad) с помощью учетных данных администратора. Можно также войти на [портал Office 365](https://portal.office.com). В меню выберите **Центры администрирования** > **Azure AD**.
2.  Создайте подписку Azure, если у вас ее нет. Если вы уже оплатили подписку, кредитная карта или оплата при этом не потребуется (щелкните ссылку подписки **Register your free Azure Active Directory** (Зарегистрировать бесплатную подписку Azure Active Directory)).
3.  Выберите **Azure Active Directory**, а затем вашу организацию.
4.  Откройте вкладку **Пользователи**.
5. Выберите пользователя, связанного с устройством, которое требуется удалить.
6.  Выберите **Устройства**.
7.  Удалите устройства. Например, можно удалить неиспользуемые или имеющие неверные определения устройства.
