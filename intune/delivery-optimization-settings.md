# <a name="delivery-optimization-settings-for-intune"></a>Параметры оптимизации доставки для Intune

В этой статье перечислены параметры оптимизации доставки, поддерживаемые Intune для устройств под управлением Windows 10 или более поздней версии.  

Большинство параметров в консоли Intune напрямую сопоставить параметры оптимизации доставки, которые входят подробные в документации по Windows, для которого предоставляются ссылки на соответствующее содержимое.  Параметры или параметры, относящиеся к Intune не содержат ссылки на дополнительное содержимое.  

Следующие таблицы содержат:  

- **Параметр**: параметр, так как он отображается в Intune. Параметры, которые являются ссылками откройте соответствующую запись в [Настройка доставки оптимизации для обновлений Windows 10](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization) в документации по Windows, где Дополнительные сведения о параметре.

- **Версия Windows**: минимальная версия Windows 10, которая включает поддержку для этого параметра.  

- **Сведения о**: краткое описание как Intune реализует параметр, в том числе по умолчанию в Intune. Если она доступна, существуют связи [поставщика службы конфигурации политики оптимизации доставки](https://docs.microsoft.com/windows/client-management/mdm/policy-csp-deliveryoptimization) записи (CSP).  

Чтобы настроить Intune для использования этих параметров, см. в разделе [доставки обновлений](delivery-optimization-windows.md).
