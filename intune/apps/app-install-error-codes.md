# <a name="intune-app-installation-error-reference"></a>Справочник по ошибкам установки приложений Intune

Помимо выполнения указанных шагов для устранения неполадок в приложениях, можно также подробнее узнать об ошибках приложений на основе возвращаемых ими кодов ошибок. После сопоставления кода ошибки воспользуйтесь дополнительным описанием и сведениями, которые помогут в устранении ошибки.
