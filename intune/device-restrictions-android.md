# <a name="microsoft-intune-android-and-samsung-knox-standard-device-restriction-settings"></a>Параметры ограничений для устройств Android и Samsung KNOX Standard в Microsoft Intune 

[!INCLUDE [azure_portal](./includes/azure_portal.md)]

В этой статье описаны все параметры ограничения устройств в Microsoft Intune, которые можно настроить для устройств под управлением Android.

>[!TIP]
>Если нужные параметры недоступны, можно настроить устройства с помощью [пользовательского профиля](custom-settings-android.md).

## <a name="general"></a>Общие

- **Камера.** Разрешает использовать камеру устройства.
- **Копирование и вставка (только для Samsung KNOX).** Разрешение на использование на устройстве функций копирования и вставки.
- **Совместное использование буфера обмена приложениями (только для Samsung KNOX).** Разрешение на использование буфера обмена для копирования и вставки между приложениями.
- **Передача диагностических данных (только для Samsung KNOX).** Запрет на передачу диагностических данных с устройства.
- **Сброс параметров до заводских настроек (только для Samsung KNOX).** Разрешение для пользователей на сброс параметров устройства к заводским значениям.
- **Геолокация (только для Samsung KNOX).** Разрешение для устройства на использование сведений о расположении.
- **Выключение питания (только для Samsung KNOX).** Разрешение для пользователя на выключение устройства.<br>Если этот параметр отключен, нельзя установить параметр **Число неудачных попыток входа перед очисткой устройства**.
- **Снимки экрана (только для Samsung KNOX).** Разрешение для пользователя на сохранение содержимого экрана в виде изображения.
- **Голосовой помощник (только для Samsung KNOX).** Разрешение на использование программного обеспечения голосового помощника на устройстве.
- **YouTube (только для Samsung KNOX).** Разрешение на использование приложения YouTube на устройстве.
- **Общие устройства (только для Samsung KNOX).** Настройка общего доступа к управляемому устройству Samsung KNOX Standard. В этом режиме конечные пользователи могут входить в систему устройства и выходить из нее, используя учетные данные Azure AD. При этом устройство остается управляемым, независимо от того, используется оно или нет.<br>При использовании в сочетании с профилем сертификата SCEP эта функция позволяет пользователям совместно использовать устройства с тем же набором приложений, но с их собственным сертификатом пользователя SCEP.  После выхода пользователя из системы все данные приложения удаляются.  Эта функция поддерживается только для бизнес-приложений.
- **Блокирование изменений даты и времени (Samsung Knox).** Запрет для пользователей на изменение даты и времени на устройстве. 

## <a name="password"></a>Пароль

- **Пароль.** Пользователь должен ввести пароль для доступа к устройству.|Да|Да|

    > [!NOTE]
    > Устройства Samsung Knox автоматически запрашивают 4-значный ПИН-код во время регистрации MDM. Собственные устройства Android могут автоматически запрашивать ПИН-код для обеспечения соответствия с условным доступом.

- **Минимальная длина пароля.** Укажите минимальную длину пароля, который должен ввести пользователь (от 4 до 16 символов).
- **Максимальное время бездействия (в минутах), по истечении которого экран блокируется.** Определяет число минут бездействия до автоматической блокировки устройства.
- **Число неудачных попыток входа перед очисткой устройства.** Определяет число разрешенных неудачных попыток входа перед очисткой устройства.
- **Окончание срока действия пароля (в днях).** Указывает количество дней до смены пароля на устройстве.
-  **Требуемый тип пароля.** Определяет требуемый уровень сложности пароля и указывает, можно ли использовать биометрические устройства. Выберите один из следующих типов.
    - **Устройство по умолчанию**.
    - **Биометрический с низким уровнем безопасности**
    - **По крайней мере числа**
    - **Числовой комплекс**. Повторяющиеся или последовательные числа (например, "1111" или "1234") не допускаются.<sup>1</sup>
    - **По крайней мере буквы**
    - **По крайней мере буквы и цифры**
    - **По крайней мере цифры, буквы и символы**
- **Запретить использование предыдущих паролей.** Запрещает повторное использование предыдущих паролей.
- **Разблокирование с помощью отпечатков пальцев (только для Samsung KNOX).** Разрешение на разблокировку поддерживаемых устройств по отпечатку пальца.
- **Smart Lock и другие доверенные агенты.** Возможность управлять функцией Smart Lock на совместных с Android устройствах (Samsung KNOX Standard 5.0 и более поздние версии). Эта функция телефона, которую иногда называют доверенным агентом, позволяет отключать или обходить пароль блокировки экрана устройства, когда устройство находится в надежном расположении. Например, это может быть подключение устройства к определенному устройству Bluetooth или его нахождение рядом с NFC-тегом. С помощью этого параметра можно запретить пользователям настраивать функцию Smart Lock.
- **Шифрование.** Требует шифровать файлы на мобильном устройстве.

    > [!NOTE]
    > Если применяется политика шифрования, пользователям устройств Samsung Knox нужно задать 6-значный сложный пароль для секретного кода устройства.

<sup>1</sup> Прежде чем назначить этот параметр устройствам, убедитесь, что на целевых устройствах используется последняя версия приложения корпоративного портала.

Если вы выбрали параметр **Сложный числовой** и назначили его устройству под управлением Android версий, предшествующих 5.0, действуют следующие правила.
- Если приложение корпоративного портала работает с версией, предшествующей 1704, политика в отношении PIN-кодов не будет применена к устройству, и на портале Azure отобразится ошибка.
- Если приложение корпоративного портала работает с версией 1704 или более поздней, могут применяться только простые ПИН-коды. Версии Android, предшествующие 5.0, не поддерживают этот параметр. Ошибка не отображается на портале Azure.


## <a name="google-play-store"></a>Магазин Google Play

- **Магазин Google Play (только для Samsung KNOX).** Разрешение для пользователей на доступ к магазину Google Play на устройстве.

## <a name="restricted-apps"></a>Ограниченные приложения

В списке приложений с ограниченным доступом можно настроить один из следующих списков одновременно для устройств Android и Samsung KNOX Standard:

**Запрещенные приложения.** Содержит приложения, не управляемые Intune, об установке и запуске которых необходимо сообщать.
**Утвержденные приложения.** Содержит приложения, которые пользователям разрешено устанавливать. Для сохранения совместимости пользователям не следует устанавливать другие приложения. Приложения под управлением Intune автоматически разрешены.
Профили устройств, содержащие параметры ограниченных приложений, должны назначаться группам пользователей.

Чтобы настроить список, щелкните **Добавить**, затем укажите выбранное имя, при необходимости издателя приложения и URL-адрес для приложения в магазине приложений.

### <a name="how-to-specify-the-url-to-an-app-in-the-store"></a>Как указать URL-адрес для приложений в магазине

Чтобы указать URL-адрес приложения в списке соответствующих и несоответствующих приложений, выполните следующие действия:

В [разделе "Приложения" Google Play](https://play.google.com/store/apps) найдите нужное приложение.

Откройте страницу установки приложения и скопируйте URL-адрес в буфер обмена. После этого его можно использовать как URL-адрес в списке приложений, соответствующих или не соответствующих политике.

Пример: найдите **Планировщик (Майкрософт)** в [разделе приложений в Google Play](https://play.google.com/store/apps). Используйте URL-адрес: **https://play.google.com/store/apps/details?id=com.microsoft.planner**.

### <a name="additional-options"></a>Дополнительные параметры

Можно также выбрать команду **Импорт** и получить список из CSV-файла. Используйте формат <*URL-адрес приложения*>, <*имя приложения*>, <*издатель приложения*> или щелкните **Экспорт**, чтобы создать CSV-файл, содержащий список приложений с ограниченным доступом в том же формате.      

## <a name="browser"></a>Браузер

- **Веб-браузер (только для Samsung KNOX).** Разрешение или запрет на использование на устройстве веб-браузера по умолчанию.
- **Автозаполнение (только для Samsung KNOX).** Разрешение на использование в веб-браузере функции автозаполнения.
- **Файлы cookie (только для Samsung KNOX).** Разрешение на использование в веб-браузере устройства файлов cookie.
- **Javascript (только для Samsung KNOX).** Разрешение на выполнение скриптов Java в веб-браузере устройства.
- **Всплывающие окна (только для Samsung KNOX).** Разрешение на блокировку всплывающего окна в веб-браузере.

## <a name="allow-or-block-apps"></a>Разрешение или блокировка приложений

Вы можете использовать эти параметры, чтобы указать, какие приложения можно устанавливать или запускать только на устройствах Samsung KNOX Standard.
Кроме того, вы можете указать, какие из установленных приложений необходимо скрыть от пользователя устройства. Пользователи не смогут запускать эти приложения.

- **Приложения, которые можно устанавливать (только на стандартных устройствах Samsung KNOX)**.
- **Приложения, которые запрещено запускать (только на стандартных устройствах Samsung KNOX)**.
- **Приложения, скрытые от пользователя (только на стандартных устройствах Samsung KNOX)**.

Настройте список приложений для каждого параметра, используя один из следующих вариантов:

- **Добавить приложения по имени пакета**. В основном используется для бизнес-приложений. Введите имя приложения и имя пакета приложения.
- **Добавить приложения по URL-адресу.** Введите имя и URL-адрес приложения в магазине Google Play.
- **Добавить управляемые приложения.** Выберите необходимое приложение в списке приложений, управляемых с помощью Intune.

## <a name="cloud-and-storage"></a>Облако и хранилище

- **Резервное копирование Google (только для Samsung KNOX).** Разрешение на резервное копирование Google.
- **Автоматическая синхронизация учетных записей Google (только для Samsung KNOX).** Разрешение на автоматическую синхронизацию параметров учетной записи Google.
- **Съемные носители (только для Samsung KNOX).** Разрешение на использование на устройстве съемных носителей, например SD-карт.
- **Шифрование для карт памяти (только для Samsung KNOX).** Указывает, обязательно ли шифровать карту памяти устройства.

## <a name="cellular-and-connectivity"></a>Сотовая сеть и подключение

- **Передача данных в роуминге (только для Samsung KNOX).** Разрешение на передачу данных в роуминге, когда устройство находится в сети сотовой связи.
- **Обмен сообщениями SMS/MMS (только для Samsung KNOX).** Разрешение на использование на устройстве SMS- и MMS-сообщений.
- **Голосовой набор (только для Samsung KNOX).** Включает или отключает функцию голосового набора на устройстве.
- **Передача голосовых данных в роуминге (только для Samsung KNOX).** Разрешение на передачу голосовых данных в роуминге, когда устройство находится в сети сотовой связи.
- **Bluetooth (только для Samsung KNOX).** Разрешение на использование Bluetooth на устройстве.
- **NFC (только для Samsung KNOX).** Разрешение на выполнение операций, использующих радиочастотную связь ближнего действия на поддерживаемых устройствах.
- **Wi-Fi (только для Samsung KNOX).** Разрешение на использование возможностей Wi-Fi на устройстве.
- **Модем Wi-Fi (только для Samsung KNOX).** Разрешение на использование модема Wi-Fi на устройстве.

## <a name="kiosk"></a>Киоск

Параметры киоска применимы только к устройствам Samsung KNOX Standard и приложениям, которые управляются с помощью Intune.

- **Выбрать управляемое приложение.** Выберите один из следующих вариантов, чтобы добавить одно или несколько управляемых приложений, которые могут выполняться на устройстве в режиме киоска. Другим приложениям не будет разрешено выполняться на устройстве. Предварительно установленные браузеры нельзя определить как приложение, которое может выполняться, пока устройство находится в полноэкранном режиме. Если требуется браузер, попробуйте использовать [Managed Browser](app-configuration-managed-browser.md).
    - **Добавить приложения по имени пакета**
    - **Добавить приложения по URL-адресу**
    - **Добавить управляемые приложения**
- **Кнопка спящего режима экрана.** Включает или отключает кнопку выхода из спящего режима экрана на устройстве.
- **Кнопки громкости**. Включает или отключает использование кнопок громкости на устройстве.
