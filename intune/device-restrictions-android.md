# <a name="android-and-samsung-knox-standard-device-restriction-settings-in-microsoft-intune"></a>Параметры ограничений для устройств Android и Samsung KNOX Standard в Microsoft Intune

[!INCLUDE[azure_portal](./includes/azure_portal.md)]

Используйте эти параметры с политикой ограниченного использования устройств Android для настройки устройств в организации.

>[!TIP]
>Если нужные параметры недоступны, можно настроить устройства с помощью [пользовательского профиля](custom-settings-android.md). 

## <a name="general"></a>Общие

- **Камера.** Разрешает использовать камеру устройства.
- **Копирование и вставка (только для Samsung KNOX).** Разрешает использовать на устройстве функции копирования и вставки.
- **Совместное использование буфера обмена приложениями (только для Samsung KNOX).** Разрешает использовать буфер обмена для копирования и вставки между приложениями.
- **Передача диагностических данных (только для Samsung KNOX).** Запрещает передавать диагностические данные с устройства.
- **Сброс параметров до заводских настроек (только для Samsung KNOX).** Разрешает пользователям сбрасывать параметры устройства к заводским значениям.
- **Геолокация (только для Samsung KNOX).** Разрешает устройству использовать сведения о расположении.
- **Выключение питания (только для Samsung KNOX).** Разрешает пользователю выключать устройство.<br>Если этот параметр отключен, нельзя установить параметр **Число неудачных попыток входа перед очисткой устройства**.
- **Снимки экрана (только для Samsung KNOX).** Разрешает пользователю сохранять содержимое экрана в виде изображения.
- **Голосовой помощник (только для Samsung KNOX).** Разрешает использовать программное обеспечение голосового помощника на устройстве.
- **YouTube (только для Samsung KNOX).** Разрешает использовать приложение YouTube на устройстве.
- **Общие устройства.** Настройка общего доступа к управляемому устройству Samsung KNOX Standard. В этом режиме конечные пользователи могут входить в систему устройства и выходить из нее, используя учетные данные Azure AD. При этом устройство остается управляемым, независимо от того, используется оно или нет.<br>Когда пользователь входит в систему, он получает доступ к приложениям, кроме того, к нему применяются соответствующие политики. После выхода пользователя из системы все данные приложения удаляются.

## <a name="password"></a>Пароль

- **Пароль.** Пользователь должен ввести пароль для доступа к устройству.|Да|Да|
- **Минимальная длина пароля.** Укажите минимальную длину пароля, который должен ввести пользователь (от 4 до 16 символов).
- **Максимальное время бездействия (в минутах), по истечении которого экран блокируется.** Определяет число минут бездействия до автоматической блокировки устройства.
- **Число неудачных попыток входа перед очисткой устройства.** Определяет число разрешенных неудачных попыток входа перед очисткой устройства.
- **Окончание срока действия пароля (в днях).** Указывает количество дней до смены пароля на устройстве.
-  **Требуемый тип пароля.** Определяет требуемый уровень сложности пароля и указывает, можно ли использовать биометрические устройства. Выберите один из следующих типов.
    - **Устройство по умолчанию**.
    - **Биометрический с низким уровнем безопасности**
    - **По крайней мере числа**
    - **Числовой комплекс.** Повторяющиеся или последовательные числа, например "1111" или "1234", не допускаются<sup>1</sup>
    - **По крайней мере буквы**
    - **По крайней мере буквы и цифры**
    - **По крайней мере цифры, буквы и символы**
- **Запретить использование предыдущих паролей.** Запрещает повторное использование предыдущих паролей.
- **Разблокирование с помощью отпечатков пальцев (только для Samsung KNOX).** Разрешает разблокировку поддерживаемых устройств по отпечатку пальца.
- **Smart Lock и другие доверенные агенты.** Позволяет управлять функцией Smart Lock на совместных с Android устройствах (Samsung KNOX Standard 5.0 и более поздние версии). Эта функция телефона, которую иногда называют доверенным агентом, позволяет отключать или обходить пароль блокировки экрана устройства, когда устройство находится в надежном расположении. Например, это может быть подключение устройства к определенному устройству Bluetooth или его нахождение рядом с NFC-тегом. С помощью этого параметра можно запретить пользователям настраивать функцию Smart Lock.
- **Шифрование.** Требует шифровать файлы на мобильном устройстве.

<sup>1</sup> Прежде чем назначить этот параметр устройствам, убедитесь, что на целевых устройствах используется последняя версия приложения корпоративного портала.

Если вы выбрали параметр **Сложный числовой** и назначили его устройству под управлением Android версий, предшествующих 5.0, действуют следующие правила.
- Если приложение корпоративного портала работает с версией, предшествующей 1704, политика ПИН-кода не будет применена к устройству и на портале Intune отобразится ошибка.
- Если приложение корпоративного портала работает с версией 1704 или более поздней, могут применяться только простые ПИН-коды. Версии Android, предшествующие 5.0, не поддерживают этот параметр. Ошибка не отображается на портале Intune.


## <a name="google-play-store"></a>Магазин Google Play

- **Магазин Google Play (только для Samsung KNOX).** Разрешает пользователю доступ к магазину Google Play на устройстве.

## <a name="restricted-apps"></a>Ограниченные приложения

В списке приложений с ограниченным доступом можно настроить один из следующих списков одновременно для устройств Android и Samsung KNOX Standard:

**Запрещенные приложения.** Содержит приложения, не управляемые Intune, которые пользователям запрещено устанавливать и запускать.
**Утвержденные приложения.** Содержит приложения, которые пользователям разрешено устанавливать. Для сохранения совместимости пользователям не следует устанавливать другие приложения. Приложения под управлением Intune автоматически разрешены.
Профили устройств, содержащие параметры ограниченных приложений, должны назначаться группам пользователей.

Чтобы настроить список, щелкните **Добавить**, затем укажите выбранное имя, при необходимости издателя приложения и URL-адрес для приложения в магазине приложений.

### <a name="how-to-specify-the-url-to-an-app-in-the-store"></a>Как указать URL-адрес для приложений в магазине

Чтобы указать URL-адрес приложения в списке соответствующих и несоответствующих приложений, выполните следующие действия:

В [разделе "Приложения" Google Play](https://play.google.com/store/apps) найдите нужное приложение.

Откройте страницу установки приложения и скопируйте URL-адрес в буфер обмена. После этого его можно использовать как URL-адрес в списке приложений, соответствующих или не соответствующих политике.

Пример: выполните поиск в Google Play по запросу "Microsoft Office Mobile". Нужный URL-адрес будет следующим: **https://play.google.com/store/apps/details?id=com.microsoft.office.officehub**.

### <a name="additional-options"></a>Дополнительные параметры

Можно также выбрать команду **Импорт** и получить список из CSV-файла. Используйте формат <*URL-адрес приложения*>, <*имя приложения*>, <*издатель приложения*> или щелкните **Экспорт**, чтобы создать CSV-файл, содержащий список приложений с ограниченным доступом в том же формате.      

## <a name="browser"></a>Браузер

- **Веб-браузер (только для Samsung KNOX).** Указывает, можно ли использовать на устройстве веб-браузер по умолчанию.
- **Автозаполнение (только для Samsung KNOX).** Разрешает использовать в веб-браузере функцию автозаполнения.
- **Файлы cookie (только для Samsung KNOX).** Разрешает использовать в веб-браузере устройства файлы cookie.
- **Javascript (только для Samsung KNOX).** Позволяет выполнять скрипты Java в веб-браузере устройства.
- **Всплывающие окна (только для Samsung KNOX).** Разрешает блокировать всплывающие окна в веб-браузере.

## <a name="cloud-and-storage"></a>Облако и хранилище

- **Резервное копирование Google (только для Samsung KNOX).** Разрешает использовать резервное копирование Google.
- **Автоматическая синхронизация учетных записей Google (только для Samsung KNOX).** Разрешает автоматическую синхронизацию параметров учетной записи Google.
- **Съемные носители (только для Samsung KNOX).** Разрешает использовать на устройстве съемные носители, например SD-карты.
- **Шифрование для карт памяти (только для Samsung KNOX).** Указывает, обязательно ли шифровать карту памяти устройства.

## <a name="cellular-and-connectivity"></a>Сотовая сеть и подключение

- **Передача данных в роуминге (только для Samsung KNOX).** Разрешает передачу данных в роуминге, когда устройство находится в сети сотовой связи.
- **Обмен сообщениями SMS/MMS (только для Samsung KNOX).** Разрешает использовать на устройстве SMS- и MMS-сообщения.
- **Голосовой набор (только для Samsung KNOX).** Включает или отключает функцию голосового набора на устройстве.
- **Передача голосовых данных в роуминге (только для Samsung KNOX).** Разрешает передачу голосовых данных в роуминге, когда устройство находится в сети сотовой связи.
- **Bluetooth (только для Samsung KNOX).** Разрешает использовать Bluetooth на устройстве.
- **NFC (только для Samsung KNOX).** Разрешает выполнение операций, использующих радиочастотную связь ближнего действия на поддерживаемых устройствах.
- **Wi-Fi (только для Samsung KNOX).** Разрешает использовать возможности Wi-Fi на устройстве.
- **Модем Wi-Fi (только для Samsung KNOX).** Разрешает использовать модем Wi-Fi на устройстве.

## <a name="kiosk"></a>Киоск

Параметры киоска применимы только к устройствам Samsung KNOX Standard.

- **Выбрать управляемое приложение.** Выберите один из следующих вариантов, чтобы добавить одно или несколько приложений, которые могут выполняться в режиме терминала устройства. Другим приложениям не будет разрешено выполняться на устройстве.
    - **Добавить приложения по имени пакета**
    - **Добавить приложения по URL-адресу**
    - **Добавить управляемые приложения**
- **Кнопка спящего режима экрана.** Включает или отключает кнопку выхода из спящего режима экрана на устройстве.
- **Кнопки громкости**. Включает или отключает использование кнопок громкости на устройстве.
